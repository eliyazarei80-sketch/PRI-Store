# PRI Store - فروشگاه حرفه‌ای ویندوز فرم

سیستم کامل مدیریت فروشگاه (POS) با رابط کاربری مدرن و امکانات کاربردی نوشته شده با **C# Windows Forms** و **SQL Server**

## امکانات
- مدیریت محصولات، مشتریان و تأمین‌کنندگان
- سبد خرید هوشمند
- فاکتورگیری + پرینت فاکتور
- گزارش‌گیری کامل فروش
- پنل ادمین با سطوح دسترسی
- دیتابیس SQL Server (فایل .mdf آماده)
