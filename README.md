# PRI Store - فروشگاه حرفه‌ای ویندوز فرم

یک سیستم کامل فروشگاهی (POS) با رابط کاربری زیبا و امکانات حرفه‌ای

### امکانات
- مدیریت محصولات، مشتریان و تأمین‌کنندگان  
- سبد خرید هوشمند  
- فاکتورگیری حرفه‌ای + پرینت فاکتور  
- گزارش‌گیری روزانه/ماهانه/سالانه  
- پنل ادمین با سطوح دسترسی  
- دیتابیس SQL Server (فایل .mdf آماده)

### تکنولوژی‌ها
- C# Windows Forms (.NET Framework)  
- SQL Server + ADO.NET  
- Guna.UI2 برای طراحی مدرن  
- Crystal Reports / FastReport

> پروژه مناسب دانشجویان، فریلنسرها و یادگیری پیشرفته WinForms
